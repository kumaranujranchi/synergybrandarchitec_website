# Synergy Brand Architect Website

This is the repository for the Synergy Brand Architect website.

## Live URL
[https://synergybrandarchitect.in/](https://synergybrandarchitect.in/)

## Project Overview
This project is a full-stack web application built with modern technologies to provide a dynamic and responsive user experience. It features a React-based frontend and an Express-based backend.

## Tech Stack

### Frontend
- **React**: UI library for building user interfaces.
- **Vite**: Next-generation frontend tooling.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Framer Motion**: Library for production-ready animations.
- **Radix UI**: Unstyled, accessible components for building high-quality design systems.
- **Wouter**: Minimalist router for React.

### Backend
- **Express**: Fast, unopinionated, minimalist web framework for Node.js.
- **Drizzle ORM**: TypeScript ORM for SQL databases.
- **PostgreSQL (Neon)**: Serverless Postgres database.
- **Passport**: Authentication middleware for Node.js.

### Other Tools & Services
- **TypeScript**: Typed superset of JavaScript.
- **Stripe**: Payment processing platform.
- **SendGrid**: Email delivery service.
- **OpenAI**: AI integration.

## Getting Started

To run this project locally, follow these steps:

1.  **Install Dependencies**
    ```bash
    npm install
    ```

2.  **Run Development Server**
    ```bash
    npm run dev
    ```
    This will start the backend server and the Vite development server.

3.  **Build for Production**
    ```bash
    npm run build
    ```

4.  **Start Production Server**
    ```bash
    npm start
    ```
